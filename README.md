##Hanoi
基本的な考え方
初期の位置(from)→　退避させておく場所(work)　→　行きたい場所の順に渡していくので(to)
1,from -> work
2,work -> to
3,work -> to

この流れが実装したい
c++,python -> 再帰
c++ -> 単純にいる場所、行きたい場所、そうでない場所を、計算でだす
python -> それぞれの場所に辞書型を使い場所の名前をつけてそれを入れ替えることで実装
